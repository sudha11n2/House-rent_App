# House Rent App

A simple House Rent application built with JavaScript, CSS and HTML.

## Description

This repository contains the source code for a small House Rent App. It demonstrates UI components and client-side logic for listing and browsing rental properties. The project is intended as a starter/demo application.

## Features

- Browse property listings
- View property details
- Responsive layout

## Tech stack

- JavaScript
- HTML
- CSS

## Prerequisites

- Node.js (recommended) — for installing dependencies and running local dev server if the project includes a tooling setup
- npm or yarn

## Installation

1. Clone the repository:

   git clone https://github.com/sudha11n2/House-rent_App.git
   cd House-rent_App

2. Install dependencies (if a package.json exists):

   npm install
   # or
   yarn install

## Running the app

- If the project uses a dev script (check package.json):

  npm start
  # or
  yarn start

- If the project is purely static (no build tools), open index.html in your browser or serve the directory with a static server:

  npx serve .

## Build

If a build script is present:

  npm run build
  # or
  yarn build
